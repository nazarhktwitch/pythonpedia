## Development

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Setup

```bash
npm install
```

### Build

```bash
npm run build
```

### Development Server

```bash
npm run serve
```

This will start a local development server at `http://localhost:8080`

### Watch Mode

```bash
npm run dev
```
