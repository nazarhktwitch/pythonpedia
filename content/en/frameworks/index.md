# Python Frameworks

Comprehensive documentation for popular Python web frameworks, including Django, Flask, FastAPI, and more

## Web Frameworks

### Django

Full-featured web framework for rapid development

**Key Features:**

- ORM (Object-Relational Mapping)
- Admin interface
- Authentication system
- URL routing
- Template engine

[Learn more about Django →](/frameworks/django/)

### Flask

Lightweight and flexible microframework

**Key Features:**

- Minimal core, extensible
- Jinja2 templating
- Werkzeug WSGI toolkit
- RESTful request handling

[Learn more about Flask →](/frameworks/flask/)

### FastAPI

Modern, fast web framework for building APIs

**Key Features:**

- High performance
- Automatic API documentation
- Type hints support
- Async/await support

[Learn more about FastAPI →](/frameworks/fastapi/)

### Pyramid

Flexible, scalable web framework

**Key Features:**

- Small core, extensible
- URL traversal
- Flexible authentication
- Support for various databases

[Learn more about Pyramid →](/frameworks/pyramid/)

### Tornado

Asynchronous web framework and networking library

**Key Features:**

- Non-blocking I/O
- Real-time web services
- WebSocket support
- High performance

[Learn more about Tornado →](/frameworks/tornado/)

### Bottle

Simple, lightweight microframework

**Key Features:**

- Single file distribution
- No dependencies
- Built-in HTTP server
- Template engine

[Learn more about Bottle →](/frameworks/bottle/)

### CherryPy

Minimalist web framework

**Key Features:**

- Object-oriented design
- Built-in HTTP server
- Plugin system
- Thread pool support

[Learn more about CherryPy →](/frameworks/cherrypy/)

### Sanic

Async web framework built for speed

**Key Features:**

- Async/await support
- High performance
- WebSocket support
- Auto-reload

[Learn more about Sanic →](/frameworks/sanic/)

### Quart

Async web framework compatible with Flask

**Key Features:**

- Flask-like API
- Async/await support
- WebSocket support
- HTTP/2 support

[Learn more about Quart →](/frameworks/quart/)

### Web2py

Full-stack web framework

**Key Features:**

- No installation required
- Built-in ticketing system
- Database abstraction layer
- Web-based IDE

[Learn more about Web2py →](/frameworks/web2py/)

## Framework Comparison

| Framework | Type | Best For | Learning Curve |
|-----------|------|---------|----------------|
| Django | Full-stack | Large applications | Moderate |
| Flask | Micro | Small to medium apps | Easy |
| FastAPI | API | REST APIs | Easy |
| Pyramid | Full-stack | Large, complex apps | Moderate |
| Tornado | Async | Real-time apps | Moderate |
| Bottle | Micro | Simple apps | Easy |
| CherryPy | Minimal | Embedded apps | Easy |
| Sanic | Async | High-performance APIs | Moderate |
| Quart | Async | Flask-like async apps | Easy |
| Web2py | Full-stack | Rapid prototyping | Easy |

## Choosing a Framework

**Choose Django if:**

- Building large, complex applications
- Need built-in admin interface
- Want batteries-included approach

**Choose Flask if:**

- Need flexibility and control
- Building small to medium applications
- Want minimal dependencies

**Choose FastAPI if:**

- Building REST APIs
- Need high performance
- Want automatic API documentation

**Choose Pyramid if:**

- Building large, scalable applications
- Need flexibility in architecture
- Want fine-grained control
