# Песочница Python

Интерактивная среда выполнения кода на базе **Pyodide** (WebAssembly).

<div class="sandbox-container" data-code="import sys
print(f'Привет из Python {sys.version}!')
print('Попробуйте написать свой код ниже и нажмите Run.')">
</div>

## Особенности

- **Без сервера**: Весь код выполняется прямо в вашем браузере.
- **Полная стандартная библиотека**: Поддерживается большинство встроенных модулей.
- **Безопасно**: Изолированная среда выполнения.
