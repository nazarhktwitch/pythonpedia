# Стандартная Библиотека Python

В стандартную библиотеку Python входит колоссальное количество модулей на все случаи жизни (часто говорят, что Python поставляется "вместе с батарейками"). Она предоставляет инструменты для работы с диском, сетью, математикой, временем и многим другим.

Ниже представлен справочник по самым важным и часто используемым встроенным модулям.

### Система и файлы

* [os](/ru/stdlib/os/) — Взаимодействие с операционной системой и путями.
* [sys](/ru/stdlib/sys/) — Системные переменные и параметры интерпретатора (например, `sys.argv`).
* [subprocess](/ru/stdlib/subprocess/) — Инструмент для запуска сторонних программ в фоне.
* [pathlib](/ru/stdlib/pathlib/) — Удобная ООП навигация по путям фалов (вместо `os.path`).
* [shutil](/ru/stdlib/shutil/) — Копирование, перенос и рекурсивное удаление папок.
* [tempfile](/ru/stdlib/tempfile/) — Безопасное создание временных папок.

### Данные и сериализация

* [json](/ru/stdlib/json/) — Чтение и запись в популярный формат JSON.
* [csv](/ru/stdlib/csv/) — Чтение и запись табличных файлов CSV.
* [sqlite3](/ru/stdlib/sqlite3/) — Встроенная база данных (SQL) прямо "из коробки" без серверов.
* [pickle](/ru/stdlib/pickle/) — Сериализация любых объектов Python в бинарные файлы.

### Математика и Время

* [datetime](/ru/stdlib/datetime/) — Базовые объекты для работы с датами и интервалами.
* [time](/ru/stdlib/time/) — Работа с секундами (Таймстемпы, паузы `sleep`).
* [calendar](/ru/stdlib/calendar/) — Расчет високосных дней, отображение календарей.
* [math](/ru/stdlib/math/) — Математические функции (корни, степени, пи, синусы).
* [random](/ru/stdlib/random/) — Генераторы случайных чисел и перемешивание списков.
* [statistics](/ru/stdlib/statistics/) — Вычисление средних значений, медиан и дисперсий.

### Алгоритмы и Структуры

* [collections](/ru/stdlib/collections/) — Идеальные контейнеры (развитые словари и очереди).
* [itertools](/ru/stdlib/itertools/) — Эффективные итераторы (бескончные циклы, комбинаторика).
* [copy](/ru/stdlib/copy/) — Глубокое рекурсивное клонирование списков, словарей, классов.
* [enum](/ru/stdlib/enum/) — Перечисления (строгие константы для кода).

### Архитектура и Типизация

* [dataclasses](/ru/stdlib/dataclasses/) — Классы хранения данных (авто-генерация кода).
* [typing](/ru/stdlib/typing/) — Модуль для строгой аннотации (подсказок) типов.
* [functools](/ru/stdlib/functools/) — Кэширование результатов, декораторы.

### Отладка и Логи

* [logging](/ru/stdlib/logging/) — Профессиональное сохранение логов и ошибок программы в файл.
* [argparse](/ru/stdlib/argparse/) — Парсинг аргументов из командной строки для скрипта.
* [traceback](/ru/stdlib/traceback/) — Извлечение и логирование стека вызовов (текста ошибки `__main__`).
* [inspect](/ru/stdlib/inspect/) — Получение "живого" текстового исходного кода запущенной функции.
* [unittest](/ru/stdlib/unittest/) — Создание наборов модульных тестов.

### Сеть и Безопасность

* [requests](/ru/stdlib/requests/) — *Сторонняя библиотека:* Идеальный и самый простой клиент для HTTP-запросов.
* [urllib](/ru/stdlib/urllib/) — Базовый функционал для разбора URL-адресов.
* [socket](/ru/stdlib/socket/) — Низкоуровневые сокеты (TCP/UDP, клиенты, серверы).
* [hashlib](/ru/stdlib/hashlib/) — Криптографические хеши (SHA-256).
* [secrets](/ru/stdlib/secrets/) — Криптографически-стойкая генерация случайных токенов для паролей.
* [uuid](/ru/stdlib/uuid/) — Генерация уникальных универсальных идентификаторов базы данных.
* [base64](/ru/stdlib/base64/) — Кодирование бинарных фото/файлов в ASCII-текст для пересылки через JSON/HTTP.

### Конкурентность

* [asyncio](/ru/stdlib/asyncio/) — Асинхронное исполнение кода (Event loop, корутины).
* [threading](/ru/stdlib/threading/) — Многопоточность (Запуск потоков ОС).
* [multiprocessing](/ru/stdlib/multiprocessing/) — Многопроцессорность.

### Текст и Архивирование

* [re](/ru/stdlib/re/) — Поиск и замена по регулярным выражениям.
* [string](/ru/stdlib/string/) — Константы английских букв и символов пунктуации.
* [zipfile](/ru/stdlib/zipfile/) — Создание и разархивирование ZIP-архивов.
