# О Pythonpedia

Pythonpedia — это портал документации для Python, размещенный на GitHub Pages. Он предоставляет полную документацию по стандартной библиотеке, популярным фреймворкам, лучшим практикам, подводным камням, примерам и стандартам кода.

## Цели проекта

- Полная, структурированная документация Python, доступная офлайн
- Интерактивное обучение с выполнением кода в браузере
- Документация с учетом версий с предупреждениями об устаревании
- Инструменты сравнения для функций, классов и фреймворков
- Документация по стандартам кода, архитектуре и правилам
- Адаптивный дизайн для мобильных и настольных устройств, доступность, SEO-оптимизация

## Участие

Мы приветствуем вклад в Pythonpedia. Пожалуйста, ознакомьтесь с нашими рекомендациями по участию на GitHub.

### Рекомендации по участию

1. Старайтесь следовать стилю кодирования PEP 8 (или другой общепринятый) для всех примеров кода
2. Используйте фактический, объективный язык (без мнений, эмодзи или ненужной пунктуации)
3. Включайте правильную информацию о версиях для функций и классов
4. Предоставляйте примеры, которые можно запустить в песочнице кода
5. Тестируйте свои изменения локально перед отправкой
